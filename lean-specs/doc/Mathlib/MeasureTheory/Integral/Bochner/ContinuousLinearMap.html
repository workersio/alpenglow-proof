    Continuous linear maps composed with integration #

    The goal of this file is to prove that integration commutes with continuous linear maps. This holds for simple functions. The general result follows from the continuity of all involved operations on the space L¹. Note that composition by a continuous linear map on L¹ is not just the composition, as we are dealing with classes of functions, but it has already been defined as ContinuousLinearMap.compLp. We take advantage of this construction here.
