    Matrices associated with non-degenerate bilinear forms #

    Main definitions #

    • Matrix.Nondegenerate A: the proposition that when interpreted as a bilinear form, the matrix A is nondegenerate.
    source
    def Matrix.Nondegenerate {m : Type u_1} {R : Type u_2} [CommRing R] [Finite m] (M : Matrix m m R) :
    Prop

    A matrix M is nondegenerate if for all v ≠ 0, there is a w ≠ 0 with w * M * v ≠ 0.

    Equations
    • M.Nondegenerate = ∀ (v : m → R), (∀ (w : m → R), v ⬝ᵥ M.mulVec w = 0) → v = 0
