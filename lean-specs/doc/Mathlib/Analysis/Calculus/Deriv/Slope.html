    Derivative as the limit of the slope #

    In this file we relate the derivative of a function with its definition from a standard undergraduate course as the limit of the slope (f y - f x) / (y - x) as y tends to ùìù[‚â†] x. Since we are talking about functions taking values in a normed space instead of the base field, we use slope f x y = (y - x)‚Åª¬π ‚Ä¢ (f y - f x) instead of division.

    We also prove some estimates on the upper/lower limits of the slope in terms of the derivative.

    For a more detailed overview of one-dimensional derivatives in mathlib, see the module docstring of analysis/calculus/deriv/basic.
