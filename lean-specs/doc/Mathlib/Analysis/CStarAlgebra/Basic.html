    Normed star rings and algebras #

    A normed star group is a normed group with a compatible star which is isometric.

    A C‚ãÜ-ring is a normed star group that is also a ring and that verifies the stronger condition ‚Äñx‚Äñ^2 ‚â§ ‚Äñx‚ãÜ * x‚Äñ for all x (which actually implies equality). If a C‚ãÜ-ring is also a star algebra, then it is a C‚ãÜ-algebra.
