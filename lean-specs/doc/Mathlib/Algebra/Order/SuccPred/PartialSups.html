    PartialSups in a SuccAddOrder #

    Basic results concerning PartialSups which follow with minimal assumptions beyond the fact that the PartialSup is defined over a SuccAddOrder.

    source
    @[simp]
    theorem partialSups_add_one {α : Type u_1} {ι : Type u_2} [SemilatticeSup α] [LinearOrder ι] [Add ι] [One ι] [LocallyFiniteOrderBot ι] [SuccAddOrder ι] (f : ι → α) (i : ι) :
    (partialSups f) (i + 1) = (partialSups f) i ⊔ f (i + 1)
