    Kernel pairs #

    This file defines what it means for a parallel pair of morphisms a b : R ‚ü∂ X to be the kernel pair for a morphism f. Some properties of kernel pairs are given, namely allowing one to transfer between the kernel pair of f‚ÇÅ ‚â´ f‚ÇÇ to the kernel pair of f‚ÇÅ. It is also proved that if f is a coequalizer of some pair, and a,b is a kernel pair for f then it is a coequalizer of a,b.

    Implementation #

    The definition is essentially just a wrapper for IsLimit (PullbackCone.mk _ _ _), but the constructions given here are useful, yet awkward to present in that language, so a basic API is developed here.
