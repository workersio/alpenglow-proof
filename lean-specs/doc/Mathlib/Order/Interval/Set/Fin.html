    (Pre)images of set intervals under Fin operations #

    In this file we prove basic lemmas about preimages and images of the intervals under the following operations:

    • Fin.val,
    • Fin.castLE (preimages only),
    • Fin.castAdd,
    • Fin.cast,
    • Fin.castSucc,
    • Fin.natAdd,
    • Fin.addNat,
    • Fin.succ,
    • Fin.rev.
