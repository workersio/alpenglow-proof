    Complex number as a finite-dimensional vector space over ℝ #

    This file contains the FiniteDimensional ℝ ℂ instance, as well as some results about the rank (finrank and Module.rank).

    source
    instance Complex.instFiniteDimensionalReal :
    FiniteDimensional ℝ ℂ
    source
    @[simp]
    theorem Complex.finrank_real_complex :
    Module.finrank ℝ ℂ = 2

    ℂ is a finite extension of ℝ of degree 2, i.e [ℂ : ℝ] = 2
