    The filter of small sets #

    This file defines the filter of small sets w.r.t. a filter f, which is the largest filter containing all powersets of members of f.

    g converges to f.smallSets if for all s ‚àà f, eventually we have g x ‚äÜ s.

    An example usage is that if f : Œπ ‚Üí E ‚Üí ‚Ñù is a family of nonnegative functions with integral 1, then saying that fun i ‚Ü¶ support (f i) tendsto (ùìù 0).smallSets is a way of saying that f tends to the Dirac delta distribution.
