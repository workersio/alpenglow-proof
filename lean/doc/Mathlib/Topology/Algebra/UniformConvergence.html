    Algebraic facts about the topology of uniform convergence #

    This file contains algebraic compatibility results about the uniform structure of uniform convergence / ùîñ-convergence. They will mostly be useful for defining strong topologies on the space of continuous linear maps between two topological vector spaces.

    Main statements #

    • UniformFun.uniform_group : if G is a uniform group, then Œ± ‚Üí·µ§ G a uniform group
    • UniformOnFun.uniform_group : if G is a uniform group, then for any ùîñ : Set (Set Œ±), Œ± ‚Üí·µ§[ùîñ] G a uniform group.

    Implementation notes #

    Like in Mathlib/Topology/UniformSpace/UniformConvergenceTopology.lean, we use the type aliases UniformFun (denoted Œ± ‚Üí·µ§ Œ≤) and UniformOnFun (denoted Œ± ‚Üí·µ§[ùîñ] Œ≤) for functions from Œ± to Œ≤ endowed with the structures of uniform convergence and ùîñ-convergence.

    References #

    • [N. Bourbaki, General Topology, Chapter X][bourbaki1966]
    • [N. Bourbaki, Topological Vector Spaces][bourbaki1987]
