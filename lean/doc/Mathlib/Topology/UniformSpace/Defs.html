    Uniform spaces #

    Uniform spaces are a generalization of metric spaces and topological groups. Many concepts directly generalize to uniform spaces, e.g.

    • uniform continuity (in this file)
    • completeness (in Cauchy.lean)
    • extension of uniform continuous functions to complete spaces (in IsUniformEmbedding.lean)
    • totally bounded sets (in Cauchy.lean)
    • totally bounded complete sets are compact (in Cauchy.lean)

    A uniform structure on a type X is a filter ùì§ X on X √ó X satisfying some conditions which makes it reasonable to say that ‚àÄ·∂† (p : X √ó X) in ùì§ X, ... means "for all p.1 and p.2 in X close enough, ...". Elements of this filter are called entourages of X. The two main examples are:

    • If X is a metric space, V ‚àà ùì§ X ‚Üî ‚àÉ Œµ > 0, { p | dist p.1 p.2 < Œµ } ‚äÜ V
    • If G is an additive topological group, V ‚àà ùì§ G ‚Üî ‚àÉ U ‚àà ùìù (0 : G), {p | p.2 - p.1 ‚àà U} ‚äÜ V

    Those examples are generalizations in two different directions of the elementary example where X = ‚Ñù and V ‚àà ùì§ ‚Ñù ‚Üî ‚àÉ Œµ > 0, { p | |p.2 - p.1| < Œµ } ‚äÜ V which features both the topological group structure on ‚Ñù and its metric space structure.

    Each uniform structure on X induces a topology on X characterized by

    nhds_eq_comap_uniformity : ‚àÄ {x : X}, ùìù x = comap (Prod.mk x) (ùì§ X)

    where Prod.mk x : X ‚Üí X √ó X := (fun y ‚Ü¶ (x, y)) is the partial evaluation of the product constructor.

    The dictionary with metric spaces includes:

    • an upper bound for dist x y translates into (x, y) ‚àà V for some V ‚àà ùì§ X
    • a ball ball x r roughly corresponds to UniformSpace.ball x V := {y | (x, y) ‚àà V} for some V ‚àà ùì§ X, but the later is more general (it includes in particular both open and closed balls for suitable V). In particular we have: isOpen_iff_ball_subset {s : Set X} : IsOpen s ‚Üî ‚àÄ x ‚àà s, ‚àÉ V ‚àà ùì§ X, ball x V ‚äÜ s

    The triangle inequality is abstracted to a statement involving the composition of relations in X. First note that the triangle inequality in a metric space is equivalent to ‚àÄ (x y z : X) (r r' : ‚Ñù), dist x y ‚â§ r ‚Üí dist y z ‚â§ r' ‚Üí dist x z ‚â§ r + r'. Then, for any V and W with type Set (X √ó X), the composition V ‚óã W : Set (X √ó X) is defined as { p : X √ó X | ‚àÉ z, (p.1, z) ‚àà V ‚àß (z, p.2) ‚àà W }. In the metric space case, if V = { p | dist p.1 p.2 ‚â§ r } and W = { p | dist p.1 p.2 ‚â§ r' } then the triangle inequality, as reformulated above, says V ‚óã W is contained in {p | dist p.1 p.2 ‚â§ r + r'} which is the entourage associated to the radius r + r'. In general we have mem_ball_comp (h : y ‚àà ball x V) (h' : z ‚àà ball y W) : z ‚àà ball x (V ‚óã W). Note that this discussion does not depend on any axiom imposed on the uniformity filter, it is simply captured by the definition of composition.

    The uniform space axioms ask the filter ùì§ X to satisfy the following:

    • every V ‚àà ùì§ X contains the diagonal idRel = { p | p.1 = p.2 }. This abstracts the fact that dist x x ‚â§ r for every non-negative radius r in the metric space case and also that x - x belongs to every neighborhood of zero in the topological group case.
    • V ‚àà ùì§ X ‚Üí Prod.swap '' V ‚àà ùì§ X. This is tightly related the fact that dist x y = dist y x in a metric space, and to continuity of negation in the topological group case.
    • ‚àÄ V ‚àà ùì§ X, ‚àÉ W ‚àà ùì§ X, W ‚óã W ‚äÜ V. In the metric space case, it corresponds to cutting the radius of a ball in half and applying the triangle inequality. In the topological group case, it comes from continuity of addition at (0, 0).

    These three axioms are stated more abstractly in the definition below, in terms of operations on filters, without directly manipulating entourages.

    Main definitions #

    • UniformSpace X is a uniform space structure on a type X
    • UniformContinuous f is a predicate saying a function f : Œ± ‚Üí Œ≤ between uniform spaces is uniformly continuous : ‚àÄ r ‚àà ùì§ Œ≤, ‚àÄ·∂† (x : Œ± √ó Œ±) in ùì§ Œ±, (f x.1, f x.2) ‚àà r

    Notation #

    Localized in Uniformity, we have the notation ùì§ X for the uniformity on a uniform space X, and ‚óã for composition of relations, seen as terms with type Set (X √ó X).

    Implementation notes #

    There is already a theory of relations in Data/Rel.lean where the main definition is def Rel (Œ± Œ≤ : Type*) := Œ± ‚Üí Œ≤ ‚Üí Prop. The relations used in the current file involve only one type, but this is not the reason why we don't reuse Data/Rel.lean. We use Set (Œ± √ó Œ±) instead of Rel Œ± Œ± because we really need sets to use the filter library, and elements of filters on Œ± √ó Œ± have type Set (Œ± √ó Œ±).

    The structure UniformSpace X bundles a uniform structure on X, a topology on X and an assumption saying those are compatible. This may not seem mathematically reasonable at first, but is in fact an instance of the forgetful inheritance pattern. See Note [forgetful inheritance] below.
