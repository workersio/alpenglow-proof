    Monads #

    We construct the categories of monads and comonads, and their forgetful functors to endofunctors.

    (Note that these are the category theorist's monads, not the programmers monads. For the translation, see the file CategoryTheory.Monad.Types.)

    For the fact that monads are "just" monoids in the category of endofunctors, see the file CategoryTheory.Monad.EquivMon.

    Type (max u‚ÇÅ v‚ÇÅ)

    The data of a monad on C consists of an endofunctor T together with natural transformations Œ∑ : ùü≠ C ‚ü∂ T and Œº : T ‚ãô T ‚ü∂ T satisfying three equations:

    • T Œº_X ‚â´ Œº_X = Œº_(TX) ‚â´ Œº_X (associativity)
    • Œ∑_(TX) ‚â´ Œº_X = 1_X (left unit)
    • TŒ∑_X ‚â´ Œº_X = 1_X (right unit)
    • obj : C ‚Üí C
    • map {X Y : C} : (X ‚ü∂ Y) ‚Üí (self.obj X ‚ü∂ self.obj Y)
    • map_id (X : C) : self.map (CategoryStruct.id X) = CategoryStruct.id (self.obj X)
    • map_comp {X Y Z : C} (f : X ‚ü∂ Y) (g : Y ‚ü∂ Z) : self.map (CategoryStruct.comp f g) = CategoryStruct.comp (self.map f) (self.map g)
    • Œ∑ : Functor.id C ‚ü∂ self.toFunctor

      The unit for the monad.

    • Œº : self.comp self.toFunctor ‚ü∂ self.toFunctor

      The multiplication for the monad.

    • assoc (X : C) : CategoryStruct.comp (self.map (self.Œº.app X)) (self.Œº.app X) = CategoryStruct.comp (self.Œº.app (self.obj X)) (self.Œº.app X)
    • left_unit (X : C) : CategoryStruct.comp (self.Œ∑.app (self.obj X)) (self.Œº.app X) = CategoryStruct.id ((Functor.id C).obj (self.obj X))
    • right_unit (X : C) : CategoryStruct.comp (self.map (self.Œ∑.app X)) (self.Œº.app X) = CategoryStruct.id (self.obj ((Functor.id C).obj X))

      The data of a comonad on C consists of an endofunctor G together with natural transformations Œµ : G ‚ü∂ ùü≠ C and Œ¥ : G ‚ü∂ G ‚ãô G satisfying three equations:

      • Œ¥_X ‚â´ G Œ¥_X = Œ¥_X ‚â´ Œ¥_(GX) (coassociativity)
      • Œ¥_X ‚â´ Œµ_(GX) = 1_X (left counit)
      • Œ¥_X ‚â´ G Œµ_X = 1_X (right counit)
      • obj : C ‚Üí C
      • map {X Y : C} : (X ‚ü∂ Y) ‚Üí (self.obj X ‚ü∂ self.obj Y)
      • map_id (X : C) : self.map (CategoryStruct.id X) = CategoryStruct.id (self.obj X)
      • map_comp {X Y Z : C} (f : X ‚ü∂ Y) (g : Y ‚ü∂ Z) : self.map (CategoryStruct.comp f g) = CategoryStruct.comp (self.map f) (self.map g)
      • Œµ : self.toFunctor ‚ü∂ Functor.id C

        The counit for the comonad.

      • Œ¥ : self.toFunctor ‚ü∂ self.comp self.toFunctor

        The comultiplication for the comonad.

      • coassoc (X : C) : CategoryStruct.comp (self.Œ¥.app X) (self.map (self.Œ¥.app X)) = CategoryStruct.comp (self.Œ¥.app X) (self.Œ¥.app (self.obj X))
      • left_counit (X : C) : CategoryStruct.comp (self.Œ¥.app X) (self.Œµ.app (self.obj X)) = CategoryStruct.id (self.obj X)
      • right_counit (X : C) : CategoryStruct.comp (self.Œ¥.app X) (self.map (self.Œµ.app X)) = CategoryStruct.id (self.obj X)
